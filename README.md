 # 📩 Contact Form

A responsive contact form with real-time validation built using **HTML, CSS, and JavaScript**.

---

## 🔗 Live Demo

👉  https://sohib-ezaldeen.github.io/Contact-form/

## 💻 Repository

👉 https://github.com/sohib-ezaldeen/contact-form

---

## ✨ Features

* Real-time validation (Live)
* Email validation using Regex
* Radio & Checkbox validation
* Error messages handling
* Success message on submit
* Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## ⚙️ How It Works

* Prevents submission if inputs are invalid
* Shows error messages instantly
* Displays success message when valid
